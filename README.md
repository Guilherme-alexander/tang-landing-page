Landing page do Tang com um mixer de sabores interativo: escolha um saquinho e veja o copo mudar de cor e sabor na hora.

🔗 **[Ver Página Web](https://guilherme-alexander.github.io/tang-landing-page/)**

## Sobre o projeto (Update 2021 > 2026)

Site institucional da marca com quatro seções: escolha de sabor, história (do laboratório à Lua), vídeo e contato. Construído do zero em HTML, CSS e JavaScript puro, sem frameworks ou dependências de build.

## Tecnologias

- HTML5 semântico
- CSS3 (custom properties, grid, animações)
- JavaScript (vanilla, sem bibliotecas)

Depois abra o `index.html` no navegador, ou sirva a pasta com uma extensão como o **Live Server** (VS Code) para recarregar automaticamente a cada alteração.

## Estrutura

```
tang-landing-page/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── script.js
└── img/
```

## Funcionalidades

- Seletor de sabor que troca a cor do tema e a imagem da jarra em tempo real
- Carrossel de destaques com troca automática e navegação por bolinhas
- Menu responsivo com versão mobile
- Formulário de contato com validação e aviso de envio
- Animações de entrada ao rolar a página
